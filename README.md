# blackjack
二十一点（Blackjack）规则
目标：手牌点数比庄家更接近 21 点，但不超过 21 点。
点数计算：
- 数字牌（2-10）按面值算
- J / Q / K 都算 10 点
- A 可以算 1 点或 11 点（自动取更有利的值）
流程：
1. 庄家发牌，每人 2 张，庄家自己 1 张明牌 + 1 张暗牌
2. 玩家按顺序决定操作：
   - 要牌（Hit）— 再拿一张
   - 停牌（Stand）— 不再要牌
   - 分牌（Split）— 两张牌相同时分成两手（额外下注）
   - 双倍（Double Down）— 下一倍注，仅拿一张牌
3. 玩家爆牌（超过 21 点）直接输
4. 玩家都结束后，庄家翻暗牌，必须 17 点以上停牌，否则一直要牌
5. 庄家爆牌则未爆的玩家赢；都不爆则比点数
特殊：玩家前两张就是 A + 10/J/Q/K = Blackjack，直接赔 1.5 倍（除非庄家也是 Blackjack，则平局）。

多人交互方案：Hot-seat 轮流操作 + 所有人牌面公开。大家围在一台电脑前，当前轮到谁谁点按钮，其他人可以看牌一起讨论——Blackjack 玩家之间不互斗，只跟庄家比，所以所有手牌公开完全没问题，反而有派对氛围。
游戏功能：
- 支持 1-4 人，可自定义名字
- 每人 1000 筹码，下注后发牌
- 要牌 / 停牌 / 双倍 / 分牌 全支持
- 庄家自动补牌至 17 点以上
- Blackjack 直接赢 1.5 倍
- 每轮结算，破产玩家出局
